Using Trello Tracking Learning Goals

Feb 27th, 2013 • Permalink

On The Rader

Technologies I’ve heard good things about but still need to decide when/if I want to learn them.

In The Queue

Technologies that I know are going to be essential to learn.

Beginner

I’ve started a toy project, or worked through some tutorialbuilding things hands-on

Intermediate

Discovered the pitfalls. Could use professionally.

On Hold

Technologies that I consider to be on the way out, so it’s not worth spending much effort getting a deeper knowledge. (That may change if I was involed in a legacy project using one of these.)

Expert

Expert doesn’t mean I’ve stopped learning.

I’ve been partly inspired by the ThoughtWorks Technology Radar which assigns Assess, Trial, Adopt and Hold.
